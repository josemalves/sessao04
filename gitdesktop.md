# GitHub Desktop

## O que é

O **GitHub Desktop** é uma aplicação gráfica oficial da GitHub
(https://github.com/apps/desktop) que permite trabalhar com repositórios Git
sem usar a linha de comandos. Disponível para Windows e macOS, integra-se
diretamente com a conta GitHub do utilizador.

## Para que serve

Permite executar as operações mais comuns do Git — criar repositórios, fazer
*commit*, mudar de *branch*, fazer *merge*, *push* e *pull* — através de uma
interface visual que mostra de forma clara as alterações feitas em cada
ficheiro.

## Funcionalidades principais

- **Histórico visual**: lista cronológica dos commits com diff lado a lado.
- **Gestão de branches**: criar, mudar, publicar e eliminar branches a partir
  de menus.
- **Sincronização**: botões `Fetch origin` e `Push origin` para puxar e enviar
  alterações.
- **Resolução de conflitos**: indicação visual dos ficheiros em conflito,
  podendo ser resolvidos no editor preferido do utilizador.
- **Integração com a conta GitHub**: autenticação OAuth, criação de
  repositórios e *pull requests* diretamente da aplicação.

## Vantagens face ao Git Bash

- Curva de aprendizagem mais suave para quem está a começar.
- Visualização imediata das diferenças em cada commit.
- Reduz a probabilidade de erros de digitação em comandos.
- Facilita operações repetitivas (commit + push) em poucos cliques.
