# GitHub Desktop

## O que é

O **GitHub Desktop** é uma aplicação gráfica oficial da GitHub
(https://github.com/apps/desktop) que permite trabalhar com repositórios Git
sem usar a linha de comandos. Disponível para Windows e macOS, integra-se
diretamente com a conta GitHub do utilizador.

## Para que serve

Permite executar as operações mais comuns do Git — criar repositórios, fazer
*commit*, mudar de *branch*, fazer *merge*, *push* e *pull* — através de uma
interface visual que mostra de forma clara as alterações feitas em cada
ficheiro.
